# 月度旅游接待人次季节时序建模与预测

## 📖 论文概述

本项目是应用时间序列分析课程的结课论文，研究月度旅游接待人次的季节性特征并建立预测模型。

### 🎯 研究目标
- 分析旅游接待人次的季节性、趋势性特征
- 建立SARIMA、Holt-Winters等时间序列预测模型
- 比较不同模型的预测性能
- 为旅游产业决策提供量化依据

### 📊 关键发现

- **数据时间跨度**：2015年1月 - 2024年12月（120个月）
- **最优模型**：SARIMA(1,1,1)(1,1,1,12)
- **预测精度**：MAPE = 1.45%，RMSE = 0.0751亿人次
- **季节特征**：明显的12个月周期性，高峰期为7-8月（暑期）、10月（国庆）、1月（春节）

## 💡 核心模型

### SARIMA(1,1,1)(1,1,1,12)

**数学形式**：
```
Φ_p(B)Φ_P(B^s)∇^d∇_s^D Y_t = Θ_q(B)Θ_Q(B^s)ε_t
```

**参数说明**：
- (p,d,q) = (1,1,1)：非季节部分
  - p=1：自回归项数
  - d=1：差分次数
  - q=1：移动平均项数
- (P,D,Q,s) = (1,1,1,12)：季节部分
  - P=1：季节自回归
  - D=1：季节差分
  - Q=1：季节移动平均
  - s=12：12个月周期

**预测精度**：
- MAE: 0.0586 亿人次
- RMSE: 0.0751 亿人次
- MAPE: 1.45%
